The Manage xls inputs page enables you to:
  1. Upload xls/xlsx files that are to be used as part of your Autodeck definitions.
  2. Replace an existing uploaded file with new content.
  3. Extend the expiration date of an existing uploaded file.
  4. Remove any uploaded files that are no longer needed on the system.

USAGE TIP If you wish to use your uploaded file to manipulate content from other pages in your generated Autodeck (Excel format) you must include the string (KeepLinks) in the worksheet name(s). This will instruct the Autodeck process to retain references to cells from the selected TBS inputs on other worksheets in your Autodeck output. Note that in some circumstances you may need to Change Source... by using the Edit Links Excel function to restablish the linkage in the generated Autodeck output file.

  1. Use the following procedure to upload a new Excel spreadsheet
      After opening the My Autodeck schedules page, select the Manage xls inputs menu option:
    • Click on the Choose file button to navigate to the required file on your system. Note that only xls and xlsx file types may be uploaded.
    • Highlight the file by selecting it and then click on Open.
    • Description: add information to help you identify this particular file when you want to include it in an Autodeck definition.
    • Click on the Upload to load your file into the BI@IBM repository or click on the Reset button to clear all your entries.
  2. Use the following procedure to replace an existing uploaded file:
      After opening the My Autodeck schedules page, select the Manage xls inputs menu option:
    • Click on the Request ID link of the file you wish to update or replace. (If you have a large portfolio of uploaded files, you may use the Search box to refine the list of displayed files by entering any matching text string.)
    • The Request ID, file name and existing description will appear in dialog boxes at the top of the page. Click on the Choose file button to navigate to the replacement file on your system.
    • Highlight the file by selecting it and then click on Open.
    • Update the Description information as required.
    • Click on the Update button to replace the file in the BI@IBM repository or click on the Reset button to clear all your entries.
Note that a warning message will be displayed if the file name you are using does not match the original. You will be prompted to confirm or cancel but be aware that if you accept the new file, you may have to adjust the Configuration template on your Autodeck definition to account for any worksheet naming or layout changes.

If the file name remains the same, you will not receive a warning message but you should still make sure that any changes to the worksheet name or layout are reflected in your Configuration template.


  3. Use the following procedure to extend the expiration date of an existing uploaded file:
      After opening the My Autodeck schedules page, select the Manage xls inputs menu option:
    • Use the checkboxes to select any file(s) that you wish to retain beyond their current expiration date. (If you have a large portfolio of uploaded files, you may use the Search box to refine the list of displayed files by entering any matching text string. You may also use the Select all checkbox).
    • Click on the Extend date button to add 30 days to the current expiration date of all selected files.
  4. Use the following procedure to delete the uploaded files that you no longer need:
      After opening the My Autodeck schedules page, select the Manage xls inputs menu option:
    • Use the checkboxes to select any file(s) that are no longer needed. (If you have a large portfolio of uploaded files, you may use the Search box to refine the list of displayed files by entering any matching text string. You may also use the Select all checkbox).
    • Click on the Delete button to remove the selected file(s) from the system.
    • Click on OK to confirm deletion or click on the Cancel button to return to the uploaded file list.
