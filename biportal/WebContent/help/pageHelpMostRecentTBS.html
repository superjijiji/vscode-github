The My recent TBS outputs page provides a simple interface to allow you to download one or more of the most recent Cognos output files from your Trigger Based Schedules.  
 
The page provides a tabular view of each of your Cognos Schedule Requests showing the following information:  
  • Report Name - the name of the Cognos report that has been scheduled.
  • e-Mail Subject - the information specified in your CSR as the e-Mail subject. Note that this is also the filename of the downloaded report output.
  • Frequency - the schedule frequency information for this request
  • Datamart / Data Load - the name of the data load used to trigger this schedule request
  • Comments - the additional information provided in your CSR to help distinguish multiple schedule requests for the same report name.
  • Last Run Date - the date on which this request was run.
  • Domain - shows the Cognos domain where this CSR was run.
  • Owner - - flag indicating whether you are the owner (Y) or the designated Back-up (B) for this Cognos Schedule Request.
  • Action - link allowing you to Download the output file if it is still available on the system. Note that output files will only be kept on the system for a maximum of two weeks and will be flagged as Not Available after this time.

You may also use the check boxes alongside each CSR to select multiple output files and then download them all as a single action by clicking on the Download all Selected button. If you want all available output files, you may click on the checkbox alongside the table column headings to select all rows. Note that the unique Request ID will be appended to the e-Mail subject to help you distinguish multiple outputs with what would otherwise be the same name. If you wish to avoid this situation, please ensure that you use unique e-Mail subjects across all of your Cognos Schedule Requests.
