The My reports - Manage folders page enables you to create and manage a series of folders to help organise the Cognos reports that you have bookmarked.  

The Manage folders page allows you to:

  • Create new folders or rename them
  • Place cognos reports into the folders you have created
  • Delete folders you have already created
  • Sequence folders

Creating a new folder

Enter the desired folder name in the text box and then click on the Create folder button to create the new folder. To rename an existing folder, highlight the folder to be renamed and then type the new name in the text box before clicking on the Rename folder button.

Moving Cognos reports into a folder

Locate the reports you wish to move, select them and then drag them to the target folder. A green check mark will indicate when they can be dropped in place. Alternatively, select the reports you want to move and then click on the Move selected button. A dialog will open prompting you to select the target destination folder.

The on-page Search function can be useful in helping identify candidate reports. Just type the desired text string in the Search box to restrict the display to all entries containing that string. Note that the search function will also find matching reports within existing folders even when closed. The Check / Uncheck all button will toggle between selecting and deselecting all displayed reports prior to initiating the Move selected request.

Deleting a folder

Highlight the folder you wish to remove and then click on the Delete folder button. Note that only the folder will be removed - the actual reports will remain on the system.

Changing the display order of folders

To re-sequence folders, drag and drop it in the new location. A green check mark will indicate when they can be dropped in place. Note that you can use the Expand all and Collapse all buttons to control how these folders appear.


 IMPORTANT NOTE - You must click on the Save changes button before clicking on the Back link to return the BI@IBM Portal page.
