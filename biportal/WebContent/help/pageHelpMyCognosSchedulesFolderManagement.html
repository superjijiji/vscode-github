The My Cognos schedules - Manage folders page enables you to create and manage a series of folders to help organise the Cognos trigger-based schedule requests (CSRs) that you have created.  

The Manage folders page allows you to:

  • Create new folders or rename them
  • Place CSRs into the folders you have created
  • Delete folders you have altready created
  • Sequence folders and associated CSRs to place them in your preferred order

Creating a new folder

Enter the desired folder name in the text box and then click on the Create folder button to create the new folder. To rename an existing folder, highlight the folder to be renamed and then type the new name in the text box before clicking on the Rename folder button.

Moving Cognos Schedule Requests into a folder

Locate the CSRs you wish to move, select them and then drag them to the target folder. A green check mark will indicate when they can be dropped in place. Alternatively, select the reports you want to move and then click on the Move selected button. A dialog will open prompting you to select the target destination folder. To help you identify the correct requests, each CSR is identified by a text string comprised of the report name, the CSR comments and the e-Mail subject separated by the | character.

The on-page Search function can be useful in helping identify candidate requests. Just type the desired text string in the Search box to restrict the display to all entries containing that string. Note that the search function will also find matching CSRs within existing folders even when closed. The Check / Uncheck all button will toggle between selecting and deselecting all displayed CSRs prior to initiating the Move selected request.

Deleting a folder

Highlight the folder you wish to remove and then click on the Delete folder button. Note that only the folder will be removed - the actual CSRs will remain on the system.

Changing the display order of folders and CSRs

To re-sequence folders or change the display order of CSRs within a folder highlight the desired content and then drag and drop it in the new location. A green check mark will indicate when they can be dropped in place. Note that you can use the Expand all and Collapse all buttons to control how these folders appear. Further changes to the order in which your Cognos Schedule Requests are displayed may be made by using the sort function in the My Cognos Schedules page. Simply click on the column headings to sort by that particular column. Note that browser cookies will retain this sort order next time you open the My Cognos schedules page


 IMPORTANT NOTE - You must click on the Save changes button before clicking on the Back link to return the My Cognos schedules page.
