The My Distribution lists page enables you to create, update and delete distribution lists that can be used alonside individual e-Mail intranet IDs as recipients for the Cognos trigger-based report schedules and Autodecks that you have created.  

accessibility

Open the page by clicking on My Distribution lists in the BI@IBM controls portlet. You may also invoke the function by using the links provided in:

  • the My Autodecks page
  • the My Cognos schedules page
  • the e-Mail information tabs in your Cognos schedule requests or your Autodeck definitions.

Creating a new distribution list

  1. Type the desired name in the Distribution List Name field making sure that you do not use spaces or '@' in this text.
  2. Type the distribution list member e-Mail intranet IDs, separated by commas, in the e-Mail intranet IDs field.
  3. Click on the Create a List button. The new distribution list will be added to the table of existing distribution lists.

Editing an existing distribution list

  1. Select the list that you want to modify by clicking on its name in the table of existing distribution lists.
  2. Modify the set of intranet IDs shown in the e_Mail intranet IDs box.
  3. Click the Update button.

Deleting a distribution list

  1. Use the checkboxes in the distribution list table to select those that you wish to delete . If you have any schedule request that is using the selected distribution list it will have a 'Yes' entry in the 'In use?' column.
  2. Click the Delete Selected List button.
  3. You will be asked to confirm your deletion request.

    Note: if the 'In use?' indicator is set to 'Yes' and the named distribution list is the only recipient for those schedule requests or e-Mailed reports, the schedule will still be run but the output will not be mailed to anyone.

Using distribution lists in your Cognos and Autodeck schedule requests

  1. Click the e-Mail information
  2. Type your distribution lists at the Enter your e-Mail address and/or distribution list(s) field. You can use both distribution lists and intranet e-Mail addresses to create your desired set of recipients but you must separate the various entries with by spaces.
  3. Complete additional information as required and then click the Submit button to save your changes
